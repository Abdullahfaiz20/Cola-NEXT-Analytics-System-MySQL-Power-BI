# 🥤 Cola NEXT Analytics System | MySQL & Power BI

An end-to-end Business Intelligence project built for a fictional beverage company, **Cola NEXT**. This project demonstrates the complete analytics workflow—from relational database design in **MySQL** to interactive business dashboards in **Power BI**—to deliver actionable insights for decision-making.

---

## 📖 Project Overview

This project showcases how to transform raw business data into meaningful insights by integrating:

- 🗄️ MySQL for database design and storage
- 📊 Power BI for data modeling and visualization
- 🔄 Power Query for data cleaning and transformation
- 📈 DAX for KPI creation and calculated measures

The analytics system covers key business domains including Customers, Orders, Products, Suppliers, Employees, and Order Items.

---

## 🎯 Project Objectives

- Design a normalized relational database.
- Build an end-to-end analytics pipeline.
- Create interactive dashboards for business reporting.
- Generate KPIs to support data-driven decision-making.

---

# 🗂 Database Structure

The database consists of six interconnected tables:

- Customers
- Products
- Orders
- Order_Items
- Suppliers
- Employees

The schema is designed using proper **Primary Keys** and **Foreign Keys** to ensure data integrity and efficient reporting.

---

# 🏗 Entity Relationship Diagram (ERD)

The relational model includes:

- Customers → Orders
- Orders → Order Items
- Products → Order Items
- Suppliers (Dimension Table)
- Employees (Dimension Table)

This structure follows best practices for analytical reporting in Power BI.
